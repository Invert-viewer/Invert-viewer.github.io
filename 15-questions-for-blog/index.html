独立博客自省问卷 15 题

说起来也好玩，本文章创建于 2025.3.3，可是直到 2025.5.4 才真正开始写

第二次更新已经是 2025.8.9 了，希望第二次更新能让我写点真正的东西罢 xwx

三编：今天是 2025.9.3，刚好咕掉了一整个月是吧 xwx

# 你的博客更新频率是多少？

更新频率？哦这简直让我难以启齿了属于是……
自从我高考完之后以来，基本上可以说是属于年更了，而且写的都是啥？年度总结？

# 你的博客上次更新是什么时候？

上次吗…… 那说实话真的很久远了，比如这篇文章的上次编辑时间是 2025.8.9。
其他文章的话，上一次的是 2024.02.04 的年终总结，然后总结了半天什么有意义的都没有写！真的很是让我惭愧。
然而，今年还是这个现状，25 年已经过去一大半了，还是只有这个还在施工中的博客自省问卷，也就是说一点点产出都没有，很惭愧。

# 你的博客文章是原创的吗？

是的，毕竟都是我自己的随笔。肯定保证原创。
而且嘛…… 就算是转载，我也不知道自己有什么好转载的……

# 你觉得自己的文章对他人有帮助吗？

咳咳…… 别说帮助了，可能就是单纯的自我感动罢了……
因为这些都是属于自己的日常，甚至可以说就是发牢骚，很难有什么帮助对于别人…… 吧？

# 你上次换博客主题 / 程序是什么时候？

上次更换主题的时间是 2025.3.3。
上一次的博客主题是 Shoka，但是问题是已经很久没有推出新的版本了，所以说更换了他的继承主题 ShokaX
说实话，还是挺想找个机会直接把主题全部推到重来的，可是 Shoka 系列的附加的小组件和依靠他们编写的文章我实在是不想放弃，就索性留下不重置了。

# 你会对博客主题进行二次开发？

二次开发？抱歉至少对于现在的我来说是天方夜谭，就最基本的前端三件套我都可以说是一窍不通，更别说是自己二次开发了。
不过说一句题外话，当时我怂恿我朋友建立的博客，他就对这个写了一篇自己的自定义主页，真的让我羡慕死了 uwu.

# 你多久打开自己博客自我陶醉一次？

自我陶醉？倒不如说是自我嘲讽 hhhhhhh
不像大部分的技术类博客，我的这里只有我的碎碎念，没什么值得我去陶醉的。
说不定呢…… 或许我能有什么机会做点什么周记？那说不定的话我可能会打开瞅几眼。
希望如此吧。

# 你近期对自己博客域名什么感受？

域名的话，最主要的是从 github.io 换成了我的朋友的服务器的域名 179.life。也是某种意义上来说好记了一些。
再加上是国内服务器，访问速度快了不少，挺棒的。

# 你每天都会看网站的流量统计吗？

那是不可能的，本来我自己就没什么经费买个自己的服务器，所以说很难看到流量统计。
再加上我自己也是基本上没怎么放在搜索引擎上推流，很难想象真的会吸引到有什么人看……

# 你通过博客的广告赚到钱了吗？

广告？赚钱？哦可以说很长一段时间来说对我是天方夜谭了。而且嘛…… 真的会有什么广告商，看得上我的博客吗？

# 你去浏览别人的博客 / 网站主要为什么？

首先我觉得最重要的是开拓视野吧，毕竟互联网上博客千千万，总有一些真的很有启发性的博客，就比如陈皓的博客，尤其推荐这一篇光是名字看起来就已经很吸引人的文章：如何超过大多数人，里面振聋发聩的文字不管让我什么时候重新看一遍都会有新收获。还有这一篇 “21 天教你学会 C++”，视角真的很独到。

其次就是寻找解决方案。在我刚刚开始搭建自己的博客的时候，也就是 2021 年。那时候像是如今流行的 Ai 对话大模型还没有被像是如今一样无孔不入。每次遇到搭建博客报错我只能一股脑的把崩溃报告塞入搜索引擎来寻求答案。然而，他们大多数都是导向一个个的博客内，这的确帮我解决不少问题。

最后，当然是欣赏别人鲜活的文字和生动的页面。即使当时的自己没法完全模仿写出来这些内容，也是给我创办和维护自己的网站提供了不小的动力。

# 看到别人分享了一篇文章，你打开第一反应是什么？

打开的第一反应应该是关注文章的内容，除非真的是那种晦涩难懂的纯技术类文章，否则我是都会看完的。
因为我觉得博客的文章既然作者敢给他发出来，在互联网上留下痕迹，那么就是有价值的。

# 你觉得博客哪方面更重要？

我感觉这就是一个取舍问题，是内容？是页面修饰？是流量？还是其他？
就以我现在博客现状来说，还是应该以内容为重。

# 近期通过写博客有哪些新收获？

最明显的感悟就是自己已经很长时间没有好好记录自己的生活了，每天都是旅人，而没有记录下自己的心情收获。
完善这一篇博客，我感觉更是对自己的检验吧。

杂谈
